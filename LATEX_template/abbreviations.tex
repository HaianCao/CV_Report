\begin{itemize}

    \item RMSE: Root Mean Square Error.
    \item SIFT: Scale-Invariant Feature Transform.
    \item SMOTE: Synthetic Minority Over-sampling Technique.
    \item SVM: Support Vector Machine.
    \item SVR: Support Vector Regression.
\end{itemize}
